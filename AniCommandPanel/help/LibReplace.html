Library Name Replace

  • Asks the user to input a search term.
  • Asks the user to input a replacement string.
  • Searches the library for the search term, then replaces all instances of it with the replacement string.
  • Note: the search term (first input) is a regular expression - if you do not know what this means, in most cases it is fine to simply input normal text to find and replace. Case sensitive.

Recommended keyboard shortcut: Ctrl+Alt+R
